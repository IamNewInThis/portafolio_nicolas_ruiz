Pet Cholito

Lead Developer Full Stack Mobile Application 2023

A comprehensive pet adoption platform designed to connect local animal shelters with potential pet owners in Latin America. The application solves the fragmentation in pet adoption by centralizing data from over 50 shelters.

Key features include real-time chat powered by WebSocket, geolocation services for reporting lost pets on a map, and secure donation processing integrated directly with Mercado Pago API.
